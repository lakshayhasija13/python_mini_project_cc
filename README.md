# Python Mini Projects

This is a repository of all Python basic projects by me (lakshayhasija13).

## Projects

- **Carly's Clippers** (`carly_clippers`) - A Python script that analyzes a hair salon's hairstyle prices and revenue using lists, calculating averages, applying discounts, and filtering affordable cuts.
- **Customer One Total** (`customer_one_total`) - A simple Python script that calculates the total cost for a customer, including items and sales tax.
- **Gradebook** (`gradebook`) - A Python script that manages a student gradebook using nested lists, supporting operations like appending subjects, updating grades, and combining semester records.
- **Len's Slice** (`len_slice`) - A Python script that manages a pizza shop's toppings and prices using lists, supporting operations like sorting, finding cheapest/priciest options, and slicing for top picks.
- **Magic 8-Ball** (`magic_8_ball`) - A Magic 8-Ball simulator that randomly generates answers to yes/no questions.
- **Physics Class** (`physics_class`) - A Python script that demonstrates physics calculations including temperature conversions (Fahrenheit/Celsius), force, energy, and work using functions.
- **Sal's Shipping** (`sal_shipping`) - A Python script that calculates shipping costs for ground, premium ground, and drone shipping based on package weight.
